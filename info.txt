Para criar um novo Model/Controller/Resource, apenas utilize:
* php artisan make:controller NomeController
* php artisan make:model NomeModel

Caso deseja criar um Model e migration:
* php artisan make:model NomeModel --migration

------------------------------------------------------------------------------------------------

Para criar uma nova Migration
* php artisan make:migration create_nome_migration_table

Antes de rodar as Migrations é necessário criar a tabela migrations no DB. (rode o comando install)
* php artisan migrate:install
* php artisan migrate
* php artisan rollback
